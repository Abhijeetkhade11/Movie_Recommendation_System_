# Movie Recommendation System

This project is a simple Movie Recommendation System built with Python. It uses the TMDB 5000 movies and credits datasets to recommend movies based on user preferences.

## Features
- Content-based movie recommendations
- Utilizes TMDB 5000 Movies and Credits datasets
- Interactive Jupyter Notebook for exploration
- Python script (`app.py`) for running the recommendation logic

## Files
- `app.py`: Main Python script for the recommendation system
- `Movie_Recommendation_System.ipynb`: Jupyter Notebook for data exploration and prototyping
- `tmdb_5000_movies.csv`: Movie metadata dataset
- `tmdb_5000_credits.csv`: Movie credits dataset
- `requirements.txt`: List of required Python packages

## Getting Started
1. Clone this repository or download the files.
2. (Optional) Create and activate a virtual environment:
   ```
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the notebook or the Python script:
   - Jupyter Notebook: `Movie_Recommendation_System.ipynb`
   - Python script: `python app.py`
